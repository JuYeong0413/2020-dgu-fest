# 2020-dgu-fest  
### :lion: 멋쟁이사자처럼과 HI-SW봉사단이 함께하는 웹 기반 전자 액자 경진 대회  
## 프로젝트 개요  
코로나19로 인해 진행되지 못한 대동제. 재학생들에게는 지난 축제의 추억을 되살려보고, 신입생들에게는 축제 분위기를 간접적으로 경험할 수 있는 기회를 제공하고자 동국대학교 멋쟁이사자처럼 8기가 모여 제작한 사이트입니다.  

## 개발환경  
- Python 3.7.3  
- Django 3.0.6
